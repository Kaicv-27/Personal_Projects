Final Grade Predictor (Python, HTML/ CSS, JavaScript)

This served as my inaugural project for my portfolio as part of the Summer Program for Incoming Students (SPIS) which I attended during summer '23 at UCSD. I and my partner, Steven Zeng, used machine learning to create a web app that takes user input from the text fields in the forms and returns a predicted final grade. The web app allows the user to choose from three algorithms (linear regression, k-nearest neighbors, and random forest) to run their prediction on, which, coupled with the links to the Google Colab notebooks where we made our models, provides a statistical analysis of the accuracy of our models.
